Run Away

Run Away With Me seeks to inspire you to lead a healthy lifestyle within a community. If you have ever found yourself wanting to go on a run but losing motivation, or wanting to exercise with someone else, Run Away With Me helps you make that idea concrete and connect with others. Whether you want to run right now, schedule a time to run later, run by the beach, etc, Run Away With me will find someone within 5 miles that matches your mile time, and will notify you via text and on the web whenever you have a match.
